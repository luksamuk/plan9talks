Plan 9 Talks
============

This repository contains certain assets for slide presentations
performed directly on Plan 9, specially through Acme.

In this directory, you'll find:

- scripts/
  Script files related to presenting slides on Acme.
  "Slide" script taken directly from https://pspodcasting.net/dan/blog/2019/plan9_desktop.html.

- unsafers5/
  Slide presentation for 5th Unsafe.rs Talks, May 1st 2021.

Have fun.
